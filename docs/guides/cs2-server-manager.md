# CS2 Server Manager

CS2 Server Manager is a companion project that provisions and manages multiple CS2 servers preconfigured for MatchZy Auto Tournament.

**Installation & usage instructions have moved to the dedicated documentation site:**

- **Quick Start guide** (recommended):  
  [https://csm.sivert.io/getting-started/quick-start/](https://csm.sivert.io/getting-started/quick-start/)

Once your servers are installed and running via CS2 Server Manager:

1. Add each server to MatchZy Auto Tournament under **Admin → Servers**.
2. Set the webhook base URL and `SERVER_TOKEN` as described in the Server Setup guide.
3. Start your tournament – server allocation, live stats, and demo uploads will work automatically.
